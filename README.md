# Fake News Detection System

This project detects whether a news article is Real or Fake using Machine Learning.

## Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression
- Gradio

## Features
- Predicts real or fake news
- Shows confidence score
- Interactive GUI using Gradio

## How to Run
1. Open the notebook
2. Run all cells
3. Enter a news article in the Gradio interface
